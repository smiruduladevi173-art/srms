# Student Record Management System (SRMS)

A desktop-based **Student Record Management System (SRMS)** developed using **Java Swing**, **SQLite**, **JDBC**, **Python Analytics**, and a **Native C Analytics Engine**.

The system provides role-based access for administrators, staff members, and students while supporting student record management, department management, subject management, marks processing, authentication, reporting, and academic analytics.

---

## Features

- **Role-Based Authentication**
- **Student Management**
- **Department Management**
- **Subject Management**
- **Marks Management**
- **SQLite Database Integration**
- **Python-Based Analytics and Chart Generation**
- **Native C Analytics Engine**
- **Academic Performance Reports**
- **Desktop Deployment using Launch4j**

---

## Technologies Used

| Layer | Technology |
|---|---|
| **Frontend** | Java Swing |
| **Navigation** | CardLayout |
| **Database** | SQLite |
| **Database Connectivity** | JDBC |
| **Security** | Argon2 Password Hashing |
| **Analytics** | Python |
| **Native Processing** | C Programming |
| **Packaging** | Launch4j |

## Installation

1. Download the latest release package from the Releases section.
2. Extract the ZIP file.
3. Run `SRMS.exe`.

---

## System Requirements

- Windows 10 or Windows 11
- Java Runtime Environment (JRE) 21 or later
- Minimum 4 GB RAM
- 500 MB Available Storage
